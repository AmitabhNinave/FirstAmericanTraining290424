4) Service:
===========
class FlightService
{
	- ListFlights()			--> HTTPClient(GetAsync)
	- SearchFlight(int id)		--> HTTPClient(GetAsync)
	- AddFlight(Flight objFlight)	--> HTTPClient(PostAsync)
	- EditFlight(Flight objFlight)	--> HTTPClient(PutAsync)
	- DeleteFlight(id flightId)	--> HTTPClient(DeleteAsync)
}
